# خطة الاختبار اليدوي الشامل

**التاريخ:** 4 يوليو 2026  
**الحالة:** جاهز للتنفيذ

---

## 📋 نظرة عامة

هذه الخطة تغطي جميع جوانب الاختبار اليدوي للمنصة بما في ذلك الفيتشرز والأداء والأمان.

---

## 🔐 اختبار المصادقة

### 1. تسجيل الدخول
- [ ] تسجيل دخول بـ email و password صحيحة
- [ ] محاولة تسجيل دخول بـ email خاطئة
- [ ] محاولة تسجيل دخول بـ password خاطئة
- [ ] تسجيل دخول بـ email غير مسجل
- [ ] التحقق من رسائل الخطأ بالعربية
- [ ] التحقق من إعادة التوجيه بعد النجاح

### 2. إنشاء حساب جديد
- [ ] إنشاء حساب بـ email و password صحيحة
- [ ] محاولة إنشاء حساب بـ email موجود
- [ ] محاولة إنشاء حساب بـ password ضعيفة
- [ ] عدم الموافقة على الشروط والأحكام
- [ ] التحقق من رسائل النجاح والخطأ
- [ ] التحقق من إعادة التوجيه بعد النجاح

### 3. استعادة كلمة المرور
- [ ] إدخال email صحيح
- [ ] إدخال email غير موجود
- [ ] التحقق من رسالة البريد الإلكتروني
- [ ] تحديث كلمة المرور من الرابط

### 4. تسجيل الخروج
- [ ] تسجيل الخروج من لوحة التحكم
- [ ] التحقق من إعادة التوجيه إلى صفحة تسجيل الدخول
- [ ] التحقق من حذف البيانات المحفوظة

## 👥 اختبار إدارة العملاء

### 1. عرض قائمة العملاء
- [ ] تحميل صفحة العملاء
- [ ] عرض قائمة العملاء الموجودين
- [ ] عرض الإحصائيات الصحيحة
- [ ] البحث عن عميل بالاسم
- [ ] البحث عن عميل برقم الهاتف
- [ ] البحث عن عميل بالبريد الإلكتروني

### 2. إضافة عميل جديد
- [ ] ملء جميع الحقول المطلوبة
- [ ] إضافة عميل بدون بريد إلكتروني
- [ ] إضافة عميل بدون عنوان
- [ ] محاولة إضافة عميل بـ اسم قصير جداً
- [ ] محاولة إضافة عميل برقم هاتف غير صحيح
- [ ] التحقق من ظهور العميل في القائمة
- [ ] التحقق من رسالة النجاح

### 3. تعديل بيانات العميل
- [ ] تعديل اسم العميل
- [ ] تعديل رقم الهاتف
- [ ] تعديل البريد الإلكتروني
- [ ] تعديل العنوان
- [ ] تعديل الحد الائتماني
- [ ] التحقق من تحديث البيانات
- [ ] التحقق من رسالة النجاح

### 4. حذف عميل
- [ ] حذف عميل موجود
- [ ] التحقق من ظهور تأكيد الحذف
- [ ] التحقق من اختفاء العميل من القائمة
- [ ] التحقق من رسالة النجاح

### 5. عرض تفاصيل العميل
- [ ] عرض جميع بيانات العميل
- [ ] عرض الفواتير المرتبطة
- [ ] عرض الرصيد الحالي
- [ ] عرض الحد الائتماني

## 🎨 اختبار الواجهة والتصميم

### 1. التخطيط
- [ ] الشريط الجانبي يظهر بشكل صحيح
- [ ] الشريط العلوي يظهر بشكل صحيح
- [ ] المحتوى الرئيسي يظهر بشكل صحيح
- [ ] الألوان متسقة

### 2. الاستجابة
- [ ] الموقع يعمل على الهاتف المحمول
- [ ] الموقع يعمل على الجهاز اللوحي
- [ ] الموقع يعمل على سطح المكتب
- [ ] لا توجد عناصر مقطوعة

### 3. سهولة الاستخدام
- [ ] الأزرار سهلة الضغط
- [ ] الحقول سهلة الملء
- [ ] الرسائل واضحة
- [ ] التنقل سهل

### 4. الدعم العربي
- [ ] جميع النصوص بالعربية
- [ ] اتجاه النص من اليمين إلى اليسار
- [ ] الأرقام تظهر بشكل صحيح
- [ ] التواريخ بالصيغة الصحيحة

## ⚡ اختبار الأداء

### 1. سرعة التحميل
- [ ] تحميل الصفحة الرئيسية < 2 ثانية
- [ ] تحميل صفحة العملاء < 2 ثانية
- [ ] تحميل صفحة الفواتير < 2 ثانية
- [ ] تحميل صفحة المنتجات < 2 ثانية

### 2. استجابة الأزرار
- [ ] الأزرار تستجيب فوراً
- [ ] لا توجد تأخيرات في الضغط
- [ ] رسائل التحميل تظهر بشكل صحيح

### 3. البحث والتصفية
- [ ] البحث يعمل بسرعة
- [ ] التصفية تعمل بسرعة
- [ ] لا توجد تأخيرات في النتائج

---

## 🔒 اختبار الأمان

### 1. المصادقة
- [ ] لا يمكن الوصول إلى الصفحات المحمية بدون تسجيل دخول
- [ ] الجلسة تنتهي عند تسجيل الخروج
- [ ] لا يمكن الوصول إلى بيانات المستخدمين الآخرين

### 2. البيانات
- [ ] البيانات محفوظة بشكل آمن
- [ ] لا يمكن تعديل بيانات المستخدمين الآخرين
- [ ] لا يمكن حذف بيانات المستخدمين الآخرين

### 3. المدخلات
- [ ] التحقق من صحة المدخلات
- [ ] عدم السماح بـ SQL Injection
- [ ] عدم السماح بـ XSS Attacks
